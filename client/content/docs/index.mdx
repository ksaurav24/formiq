## Installation

```npm
npm install formiq-sdk
```

## Get Your Credentials

Visit the [FormIQ Dashboard](https://formiq.devxsaurav.in/dashboard) and:

1. Create a new project
2. Copy your **API Key**
3. Copy your **Project ID**

## Basic Usage

```js lineNumbers
import Formiq from 'formiq-sdk';

// Initialize the client
const formiq = new Formiq('YOUR_API_KEY', 'YOUR_PROJECT_ID');

// Submit form data
const response = await formiq.submitForm({
  name: 'John Doe',
  email: 'john@example.com',
  message: 'Hello from FormIQ!'
});

console.log('Form submitted:', response);
```

```js lineNumbers
import Formiq from 'formiq-sdk';

// Initialize FormIQ client
const formiq = new Formiq('YOUR_API_KEY', 'YOUR_PROJECT_ID');

// Get form elements
const form = document.getElementById('contactForm');
const submitBtn = document.getElementById('submitBtn');
const messageDiv = document.getElementById('messageDiv');

// Handle form submission
form.addEventListener('submit', async (e) => {
  e.preventDefault();
  
  // Disable button and show loading state
  submitBtn.disabled = true;
  submitBtn.textContent = 'Submitting...';
  messageDiv.innerHTML = '';
  
  // Get form data
  const formData = new FormData(form);
  const data = Object.fromEntries(formData);
  
  try {
    // Submit to FormIQ
    const response = await formiq.submitForm(data);
    
    // Show success message
    messageDiv.innerHTML = '<div class="message success">Form submitted successfully!</div>';
    form.reset();
    
    console.log('Success:', response);
  } catch (error) {
    // Show error message
    messageDiv.innerHTML = `<div class="message error">Error: ${error.message}</div>`;
    
    console.error('Error:', error);
  } finally {
    // Re-enable button
    submitBtn.disabled = false;
    submitBtn.textContent = 'Submit';
  }
});
```

## Environment Variables

For production, store your credentials securely:

```js lineNumbers
export const FORMIQ_CONFIG = {
  apiKey: process.env.FORMIQ_API_KEY || 'YOUR_API_KEY',
  projectId: process.env.FORMIQ_PROJECT_ID || 'YOUR_PROJECT_ID'
};
```

```js lineNumbers
import Formiq from 'formiq-sdk';
import { FORMIQ_CONFIG } from './config.js';

const formiq = new Formiq(
  FORMIQ_CONFIG.apiKey,
  FORMIQ_CONFIG.projectId
);
```
