# Installation

Get FormIQ SDK installed and create your first client instance.

## Install Package

```npm
npm install formiq-sdk
```

<Callout type="info">
FormIQ SDK works with Node.js 16+ and all modern browsers.
</Callout>

---

## Get Your Credentials

<Steps>
  <Step>
    Visit the [FormIQ Dashboard](https://formiq.devxsaurav.in/dashboard)
  </Step>
  <Step>
    Create a new project or select an existing one
  </Step>
  <Step>
    Copy your **API Key** and **Project ID** from the project settings
  </Step>
</Steps>

## Create FormIQ Instance

### Basic Setup

```ts lineNumbers
import Formiq from 'formiq-sdk';

const formiq = new Formiq('YOUR_API_KEY', 'YOUR_PROJECT_ID');
```

### With Environment Variables

**Recommended for production:**

```ts lineNumbers
import Formiq from 'formiq-sdk';

const formiq = new Formiq(
  process.env.FORMIQ_API_KEY,
  process.env.FORMIQ_PROJECT_ID
);
```

## TypeScript Setup

FormIQ includes TypeScript definitions out of the box:

```ts lineNumbers
import Formiq from 'formiq-sdk';

const formiq: Formiq = new Formiq(
  process.env.FORMIQ_API_KEY!,
  process.env.FORMIQ_PROJECT_ID!
);

// Full type inference available
const response = await formiq.submitForm({
  name: 'John Doe',
  email: 'john@example.com'
});
```

## Verify Installation

Test your setup with a simple submission:

```ts lineNumbers
import Formiq from 'formiq-sdk';

const formiq = new Formiq('YOUR_API_KEY', 'YOUR_PROJECT_ID');

// Test submission
async function testSetup() {
  try {
    const response = await formiq.submitForm({
      test: 'Hello FormIQ!',
      timestamp: new Date().toISOString()
    });
    
    console.log('✓ Setup successful!');
    console.log('Response:', response);
  } catch (error) {
    console.error('✗ Setup failed:', error.message);
  }
}

testSetup();
```

## Multiple Projects

You can create multiple instances for different projects:

```ts lineNumbers
import Formiq from 'formiq-sdk';

// Contact form
const contactFormiq = new Formiq(
  process.env.CONTACT_API_KEY,
  process.env.CONTACT_PROJECT_ID
);

// Newsletter
const newsletterFormiq = new Formiq(
  process.env.NEWSLETTER_API_KEY,
  process.env.NEWSLETTER_PROJECT_ID
);

// Use them separately
await contactFormiq.submitForm({ name: 'John', email: 'john@example.com' });
await newsletterFormiq.submitForm({ email: 'subscriber@example.com' });
```
